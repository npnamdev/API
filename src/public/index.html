API Endpoints

Authentication

  • GET /api/auth/me - Thông tin người dùng hiện tại

Courses

  • GET /api/courses - Lấy danh sách khóa học
  • GET /api/courses/:slug - Lấy khóa học theo slug
  • GET /api/courses/:id - Lấy khóa học theo ID

Chapters

  • GET /api/chapters - Lấy danh sách chương
  • GET /api/chapters/:id - Lấy chương theo ID

Lessons

  • GET /api/lessons - Lấy danh sách bài học
  • GET /api/lessons/:id - Lấy bài học theo ID

Users

  • GET /api/users - Lấy danh sách người dùng
  • GET /api/users/:id - Lấy người dùng theo ID

Categories

  • GET /api/categories - Lấy danh sách danh mục
  • GET /api/categories/:id - Lấy danh mục theo ID

Notifications

  • GET /api/notifications - Lấy danh sách thông báo

Automations

  • GET /api/automations - Lấy danh sách automations
  • GET /api/automations/:id - Lấy automation theo ID

Carts

  • GET /api/carts - Lấy giỏ hàng

Activation Codes

  • GET /api/activation-codes - Lấy danh sách mã kích hoạt
  • GET /api/activation-codes/:id - Lấy mã kích hoạt theo ID

Items

  • GET /api/items - Lấy danh sách items
  • GET /api/items/:id - Lấy item theo ID

Lesson Progress

  • GET /api/lesson-progress/last/:userId - Lấy tiến trình bài học cuối cùng

Media

  • GET /api/media - Lấy danh sách media
  • GET /api/media/:id - Lấy media theo ID

Permissions

  • GET /api/permissions - Lấy danh sách quyền
  • GET /api/permissions/:id - Lấy quyền theo ID

Roles

  • GET /api/roles - Lấy danh sách vai trò
  • GET /api/roles/:id - Lấy vai trò theo ID
  • GET /api/roles/:id/permissions - Lấy quyền của vai trò

Sections

  • GET /api/sections - Lấy danh sách sections
  • GET /api/sections/:id - Lấy section theo ID

Statistics

  • GET /api/statistics - Lấy thống kê

Topics

  • GET /api/topics - Lấy danh sách topics
  • GET /api/topics/:id - Lấy topic theo ID

User Groups

  • GET /api/user-groups - Lấy danh sách nhóm người dùng
  • GET /api/user-groups/:id - Lấy nhóm người dùng theo ID

Course Materials

  • GET /api/courses/:courseId/materials - Lấy tài liệu khóa học

Course Notifications

  • GET /api/courses/:courseId/announcements - Lấy thông báo khóa học

Contacts

  • GET /api/contacts - Lấy danh sách liên hệ
  • GET /api/contacts/:id - Lấy liên hệ theo ID

Other

  • GET /api/enrollments - Lấy danh sách đăng ký
  • GET /api/enrollments/user/:userId - Lấy đăng ký theo user
  • GET /api/enrollments/my-courses - Lấy khóa học của tôi
  • GET /api/users/:userId/courses - Lấy khóa học của user
  • GET /api/hero-banner - Lấy hero banner
  • GET /api/faq - Lấy danh sách FAQ
  • GET /api/testimonials - Lấy danh sách lời chứng thực
