API Endpoints

Authentication

  • POST /api/auth/register - Đăng ký tài khoản
  • POST /api/auth/login - Đăng nhập
  • POST /api/auth/logout - Đăng xuất
  • GET /api/auth/me - Thông tin người dùng hiện tại

Courses

  • GET /api/courses - Lấy danh sách khóa học
  • GET /api/courses/:slug - Lấy khóa học theo slug
  • GET /api/courses/:id - Lấy khóa học theo ID
  • POST /api/courses - Tạo khóa học mới
  • PUT /api/courses/:id - Cập nhật khóa học
  • DELETE /api/courses/:id - Xóa khóa học

Chapters

  • GET /api/chapters - Lấy danh sách chương
  • GET /api/chapters/:id - Lấy chương theo ID
  • POST /api/chapters - Tạo chương mới
  • PUT /api/chapters/:id - Cập nhật chương
  • DELETE /api/chapters/:id - Xóa chương

Lessons

  • GET /api/lessons - Lấy danh sách bài học
  • GET /api/lessons/:id - Lấy bài học theo ID
  • POST /api/lessons - Tạo bài học mới
  • PUT /api/lessons/:id - Cập nhật bài học
  • DELETE /api/lessons/:id - Xóa bài học

Users

  • GET /api/users - Lấy danh sách người dùng
  • GET /api/users/:id - Lấy người dùng theo ID
  • PUT /api/users/:id - Cập nhật người dùng
  • DELETE /api/users/:id - Xóa người dùng

Categories

  • GET /api/categories - Lấy danh sách danh mục
  • GET /api/categories/:id - Lấy danh mục theo ID
  • POST /api/categories - Tạo danh mục mới
  • PUT /api/categories/:id - Cập nhật danh mục
  • DELETE /api/categories/:id - Xóa danh mục

Notifications

  • GET /api/notifications - Lấy danh sách thông báo
  • POST /api/notifications - Tạo thông báo mới
  • PATCH /api/notifications/:id/read - Đánh dấu đã đọc
  • DELETE /api/notifications/:id - Xóa thông báo

Other

  • GET /api/enrollments - Lấy danh sách đăng ký
  • GET /api/orders - Lấy danh sách đơn hàng
  • GET /api/faq - Lấy danh sách FAQ
  • GET /api/testimonials - Lấy danh sách lời chứng thực
